### Anchor Creation Function
```bash
create_anchor() {
    local text="$1"
    echo "$text" | tr '[:upper:]' '[:lower:]' | \
    sed 's/[^a-zA-Z–∞-—è—ë0-9 ]//g' | \
    sed 's/[[:space:]]\+/-/g' | \
    sed 's/--*/-/g' | \
    sed 's/^-\|-$//g'
}
```
